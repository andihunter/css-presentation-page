CSS Topic - Opacity and rgba()

Opacity and rgba() styling are simple styling commands that are compatible with every browser. Below you will find the definitions of each and the syntax for the css code via CodePen.

Opacity

Opacity is how transparent an object (image, svg, etc.) is with 1 being not transparent, .5 being 50% transparent, and 0 being totally transparent. Opacity should be used when you want a whole <div>'s opacity to be changed (including text and background). The opacity function can be manipulated using numbers (0-1) or percentages.

See the Pen Opacity Example by Andrea Jensen Hunter (@andihunter) on CodePen.

rgba()

rgba() is a function that allows one to determine color and degree of transparencey. R (red), G (green), B (blue) levels are determined by numbers or percentages (255 = 100%). A (alpha = transparency) is also determined by numbers (0-1) or percentages. rgba() should be used when you want only one element of a <div> to be changed (e.g., just the text).
